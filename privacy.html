Babji Privacy Policy

Last updated: March 28, 2026

1. Overview

Babji is a personal AI assistant that helps users manage routines, reminders, goals, and daily context. This policy explains what information Babji may process, how it is used, and how users can control it.

2. Information Babji may process

Depending on user permissions and feature settings, Babji may process:

  • Account information such as display name and sign-in details
  • Tasks, reminders, nutrition entries, and user-provided messages
  • Notification interactions
  • Optional location data for place and routine learning
  • Optional Screen Time or routine timing signals used to support morning and nightly prompts
  • Device context such as timezone, locale, and app activity state

3. How Babji uses information

Babji uses data to:

  • Respond to user requests and power assistant features
  • Generate personalized reminders and routine suggestions
  • Learn approximate wake and closeout patterns when enabled by the user
  • Improve user-facing relevance and reduce generic notifications

4. Consent and controls

Babji is designed around explicit user consent. Users can disable learning-related features, notification behavior, Screen Time usage, and location access in the app or through iOS settings.

5. Notifications and routine signals

If enabled, Babji may use consented routine signals to estimate when a user starts and ends the day. These features are intended to deliver one relevant morning greeting and one nightly closeout, rather than repeated default notifications.

6. Data sharing

Babji does not sell personal data. Data may be processed by infrastructure or service providers necessary to operate the app, subject to applicable privacy and security obligations.
